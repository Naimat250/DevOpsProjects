Welcome to My Dockerized Multi-stage Flask App!

This Flask web application is automatically built, tested, and deployed using Jenkins CI/CD pipeline.

It includes:

  • Automated cloning of the GitHub repository
  • Code analysis with SonarQube for quality and security checks
  • Validation against SonarQube quality gates
  • Management of Docker containers with Docker Compose
  • Security scanning of Docker images using Trivy
  • Dependency checks for security vulnerabilities with OWASP Dependency Check
  • Deployment of Docker images to Docker Hub
